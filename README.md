# MLOps Mini Projects

A collection of hands-on MLOps practice projects — built to learn and apply production-style machine learning workflows, from data pipelines and model training to containerization, CI/CD, and deployment.

Each project in this repository lives in its own folder and focuses on a different real-world use case, while practicing core MLOps concepts end to end.

## 🎯 Purpose

This repository serves as a practical playground for core MLOps concepts, including:
- Structuring ML code into modular, reusable pipeline components
- Data ingestion, validation, and transformation
- Model training, evaluation, and versioning
- Containerization with Docker
- CI/CD automation for ML workflows
- Moving from notebook experimentation to production-style pipelines

## 🛠️ Tech Stack

| Category | Tools |
|---|---|
| Language | Python |
| ML | scikit-learn, Pandas, NumPy |
| Experimentation | Jupyter Notebook |
| Experiment Tracking | MLflow |
| Data & Model Versioning | DVC |
| Model Serving | FastAPI |
| Containerization | Docker |
| CI/CD | GitHub Actions |
| Cloud | AWS |
| Version Control | Git & GitHub |

*(Individual projects may use additional tools — see each project's own folder for specifics.)*

## 🚀 Getting Started

\`\`\`bash
# Clone the repository
git clone https://github.com/trijesh61/MLops-mini_projects.git
cd MLops-mini_projects

# Navigate into a project folder
cd "<project-folder-name>"

# Create and activate a virtual environment
python -m venv venv
source venv/bin/activate   # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
\`\`\`

Each project folder is self-contained with its own setup and run instructions.
